# MAT2CSV
Transfer .mat files to .csv files.


    There are programs to process .mat files and .csv files.
    The main function 'MATDataProcess' is able to transfer .mat files to .csv files on a computer without MATLAB.
    When you have .csv files transfered, you can select variables to plot curves using the program 'PlotwithCSV'.
    If you want merge the first row and the second row in your .csv files, you maybe need the program 'MergeHeader'.
    If you have .csv files divided and you want merge them by name, you can use the program 'MergeCSV'.
